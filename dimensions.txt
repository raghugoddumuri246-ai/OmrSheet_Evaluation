3. Box Grid (OmrBoxGrid)
File: src/components/OMR/OmrBoxGrid/OmrBoxGrid.jsx & .module.css

Purpose: The OmrBoxGrid component is used for numeric input fields in the OMR sheet, typically for:

Roll numbers
Student IDs
Any numeric identification
Integer answers (as seen in the IntegerOmr component)
Visual Representation:

[ ][ ][ ][ ][ ][ ]  (6 boxes by default)
Key Properties:

Box Dimensions (20px Ã— 28px): Taller than wide to accommodate digits
Horizontal Layout: Numbers flow from left to right
Borders: Only left borders are shown (except first box) for clean separation
Configurable Count: Defaults to 6 boxes but can be customized via props
Usage in OMR Sheet:

For Student Identification:
Used to input roll numbers or student IDs
Each box represents one digit (0-9)
Example: [1][2][3][4][5][6] for student ID 123456
For Integer Answers:
Used in the IntegerOmr component
Each box can be filled with a single digit
Example for answer "42": [ ][ ][ ][ ][4][2]
For Test Codes:
Can be used to input test/paper codes
Helps in identifying different test versions
